Voici la méthode la plus simple pour mettre ton projet sur GitHub 👇

🟩 1. Crée ton dépôt sur GitHub

  1. Va sur : https://github.com

  2. Clique en haut à droite sur New repository (ou “Nouveau dépôt”).

  3. Donne un nom à ton dépôt (ex : mon-projet-bibliotheque).

  4. Laisse tout par défaut et clique sur Create repository.

🟦 2. Ouvre ton projet sur ton PC

Va dans le dossier où ton projet est enregistré (HTML/CSS/JS, PHP, peu importe).

🟪 3. Ouvre Git Bash dans ce dossier

Dans le dossier → clic droit → Git Bash Here
(si tu as Git installé, ça apparaît automatiquement)

🟥 4. Tape ces commandes dans Git Bash

🔹 Initialiser git :

bash
git init

🔹 Ajouter tous tes fichiers :

bash
git add .

🔹 Faire ton premier commit :

bash
git commit -m "Premier commit"

🔹 Lier ton dossier au dépôt GitHub que tu as créé :

(remplace monlien par l’URL donnée par GitHub, ex : https://github.com/username/projet.git)

bash
git remote add origin monlien

🔹 Envoyer ton projet sur GitHub :

bash
git push -u origin main

Si "main" ne marche pas, essaie :

bash
git push -u origin master

🟩 5. C’est bon, ton projet apparaît sur GitHub 🎉

actualise la page GitHub → ton code est là.
